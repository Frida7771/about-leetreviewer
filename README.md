## Introduction

LeetReviewer helps you retain what you solve on LeetCode through:

- **Smart review scheduling** — automatically plans the next best time to revisit each problem, based on a spaced-repetition rhythm tuned for long-term memory.
- **Automatic LeetCode problem tracking** — log a problem once, and LeetReviewer keeps it on your review queue until you truly own it.
- **Curated problem sets** — built-in support for **LeetCode Hot 100**, **Blind 75**, and **NeetCode 150**, with per-category progress tracking.
- **Progress statistics** — visualize your weekly solved / reviewed problems, difficulty breakdowns, and streaks.
- **Bilingual UI** — English and 中文 available out of the box.

Visit [leetreviewer.com](https://leetreviewer.com) to start using LeetReviewer or view the documentation.
